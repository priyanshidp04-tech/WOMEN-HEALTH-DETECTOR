# Women Health Detector 🌸🤖

Women Health Detector is a beginner-friendly AI-based web application designed to identify hidden health issues in women through text input, voice description, and facial emotion analysis.

The project helps detect signs of stress, fatigue, emotional distress, pain indicators, and possible hormonal imbalance using simple AI logic and pattern matching.

It is built using Python, Flask, HTML, CSS, and JavaScript, making it easy to understand and extend for beginners.

## Key Features

* Chat-based symptom analysis
* Voice stress detection
* Facial emotion detection
* Silent health risk monitoring
* Health report generation

## Goal

To provide early digital health awareness and encourage timely medical attention through simple AI support.
